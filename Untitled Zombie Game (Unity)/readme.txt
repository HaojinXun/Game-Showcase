Game Title: Untitled Zombie Game

Environment: Unity

Date: 2023 Jan-March

The game was developed in a group of three with Scrum principles.

Haojin Xun was mainly responsible for the design of the game, state machine of different units, animations, and character behaviors according to the player's mouse input in Unity.
